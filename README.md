# AscInput

AscInput is a Input Manager for [OpenSiv3D](https://github.com/Siv3D/OpenSiv3D).  
The following inputs (key and axis) are supported.  

- Keyboard
- Gamepad
- XInput

***

AscInputは [OpenSiv3D](https://github.com/Siv3D/OpenSiv3D) 用の入力マネージャーです。  
以下の入力をサポートしています。  

- キーボード
- ゲームパッド
- XInput
